# emoji-term

![Screenshot](screenshot.png)

## Installation

```bash
git clone https://github.com/xxdavid/emoji-term
cd emoji-term
./install.rb
echo 'export PATH=$PATH:~/emoji-term/bin' >> ~/.zshrc
```

Only tested on macOS.
